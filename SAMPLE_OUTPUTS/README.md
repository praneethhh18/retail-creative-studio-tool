# Sample Outputs

This folder contains example exported creatives from the Retail Media Creative Tool.

## Directory Structure

```
SAMPLE_OUTPUTS/
├── facebook_feed/
│   ├── creative_summer_sale.jpg
│   └── creative_new_product.jpg
├── instagram_feed/
│   ├── creative_summer_sale.jpg
│   └── creative_new_product.jpg
├── instagram_story/
│   ├── creative_summer_sale.jpg
│   └── creative_new_product.jpg
└── instore_a4/
    ├── creative_summer_sale.jpg
    └── creative_new_product.pdf
```

## Output Specifications

### Facebook Feed
- Dimensions: 1200×628px
- Format: JPEG
- Max file size: 500KB

### Instagram Feed
- Dimensions: 1080×1080px
- Format: JPEG/PNG
- Max file size: 500KB

### Instagram Story
- Dimensions: 1080×1920px
- Format: JPEG/PNG
- Max file size: 500KB

### In-Store A4
- Dimensions: 2480×3508px (300 DPI)
- Format: JPEG/PDF
- Includes bleed area

## Quality Notes

- All exports are optimized for their target platform
- File sizes are kept under 500KB while maintaining quality
- Colors are in sRGB color space for web compatibility
- Print outputs (A4) use CMYK-safe colors
