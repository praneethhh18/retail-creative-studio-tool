# Sample Assets

This folder contains sample assets for testing and demonstration purposes.

## Packshots

Place product packshot images here. Requirements:
- Format: PNG (transparent background preferred) or JPG
- Minimum resolution: 500x500px
- Maximum file size: 10MB

Example files:
- `packshot_cereal.png` - Breakfast cereal box
- `packshot_drink.png` - Beverage bottle
- `packshot_snack.png` - Snack package

## Logos

Place brand and retailer logos here. Requirements:
- Format: PNG (transparent background required)
- Minimum resolution: 200x200px
- Include variations: horizontal, stacked, icon-only

Example files:
- `logo_brand.png` - Your brand logo
- `logo_tesco.png` - Tesco retailer tag (if applicable)

## Backgrounds

Place background images here. Requirements:
- Format: JPG or PNG
- Recommended resolution: 2000x2000px or higher
- Consider: lifestyle, abstract, solid colors

Example files:
- `background_summer.jpg` - Summer lifestyle scene
- `background_kitchen.jpg` - Kitchen/home setting
- `background_abstract.jpg` - Abstract color gradient

## Usage

1. Upload assets via the web interface
2. Or copy directly to `backend/uploads/` for testing
3. Assets will be processed (background removal, color extraction)

## Licensing

Ensure you have rights to use any images for your intended purpose.
Sample assets should be:
